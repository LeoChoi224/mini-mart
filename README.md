## 🎯 프로젝트 목표

---

판매자 : 상품 등록, 수정, 삭제 등 서비스 구현  
구매자 : 상품 검색, 장바구니 담기 등 서비스 구현

---

- Gradle
- JPA
- Junit5
- Mysql
- Redis
- Mockito
